## C语言程序
一个C语言程序，不管其大小如何，都是由函数和变量组成的。
函数中包含一些语句，以指定所要执行的计算操作；
变量，则用于存储计算过程中使用的值；

## 函数
函数应该简短而漂亮，并且只完成一件事情。函数应该可以一屏或者两人屏显示完，
只做一件事情，而且把它做好。
